    - Arrays
    - Multidimensional Arrays
    - Conditions
    - Strings
    - For loops
    - While loops
    - Functions
    - Static

- **Advanced**
    - Memory Management
    - File I/O
    - Preprocessor
    - Pointers
    - Structures
    - Function arguments by reference
    - Dynamic allocation
    - Arrays and Pointers
    - Recursion
    - Linked lists
    - Binary trees
    - Unions
    - Pointer Arithmetics
    - Function Pointers
    - Bitmasks
